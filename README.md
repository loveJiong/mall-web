# 项目简介

> 传统电商web版

## 技术栈

> vue2 + vuex + vue-router + webpack + ES6 + sass + flex

## 项目运行
```
git clone https://github.com/loveJiong/mall-web.git

cd mall-web

npm install

npm run dev
```
### 感谢 [Exrick](https://github.com/Exrick) 的开源 [xmall-front](https://github.com/Exrick/xmall-front) 项目提供前端页面及框架支持
